Sobre esta extensión
BBCodeXtra is an extension, compatible with Mozilla FireFox and Seamonkey, which adds to the context menu new commands to insert BBCode/Html/XHtml/Markdown code in an easy and fast way.

BBCodeXtra suits well with all forum types and some rich text ares, for example:
  • phpBB
  • Invision
  • VBullettin
  • GitHub

This extension can really simplify your life with all forums that use BBCode or Html codes when posting new messages.
