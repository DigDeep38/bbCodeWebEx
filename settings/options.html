This program works by evaluating tokens.

The basic tokens are:

Tag Syntax (Case significant) Example (using bbCode, just because)
 Selection  {{select}}  [b]{{select}}[/b] Bolds selected text
 Clipboard  {{clipboard}}  [b]{{clipboard}}[/b] Pastes clipboard text as bold
 List  {{list,text,type}} {{list,{{clipboard}},bbcode}}
Paste text as a bbCode unordered list. See below for current list types.
Popup {{popup,title,start text, end text}} [quote{{popup,"Enter Author","=\"","\""]{{clipboard}}[/quote]
Pasts the clipboard as a quote, and prompt for author name.  If left blank, then it is a simple "[quote]" tag.

Advanced

List types:

bbCode
  • bbCode Unordered (dots): bbcode
  • bbCode Ordered (numbers): bbcodeord
  • bbCode Alpha (alpha): bbcodeordalf
HTML (XHTML has identical syntax for lists)
  • HTML Unordered (dots): html
  • HTML Ordered (number): htmlord
  • HTML Ordered (alpha): htmlordalf
Markdown
  • Markdown Unordered (dots): markdown
  • Markdown Ordered (number): markdownord
Quote marks: in thise circumstances where you need to put in a quote mark as an argument, you can escape the quote mark by prepending a backslash "<a href=\"", or you can use a single quote, '<a href="'.
